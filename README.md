# NayePankh Volunteer AI Agent

## Overview

NayePankh Volunteer AI Agent is an AI-powered volunteer management system designed to help NGOs streamline volunteer registration, role recommendations, engagement tracking, and information assistance.

The system enables organizations to register volunteers, manage volunteer information, analyze volunteer interests, and provide instant responses to common questions.

---

## Features

### Volunteer Registration

* Register volunteers with name, email, and area of interest
* Input validation
* Automatic role recommendations

### Volunteer Dashboard

* View registered volunteers
* Organized volunteer records

### Analytics

* Volunteer statistics
* Interest distribution visualization
* Pie chart reporting

### AI Volunteer Assistant

* Answers questions about:

  * NayePankh Foundation
  * Volunteer opportunities
  * Internship programs

### Database Integration

* SQLite-based storage
* Persistent volunteer records
